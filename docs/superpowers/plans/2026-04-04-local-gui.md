# Local GUI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a lightweight local desktop GUI that runs the existing translation engine without replacing the CLI, so users can choose either terminal workflows or a beginner-friendly windowed interface.

**Architecture:** Keep the CLI intact and introduce a separate GUI entry point backed by the same service layer. Extract shared run orchestration out of `cli.py`, then build a `tkinter` GUI that prepares configs, runs pipelines on a background thread, streams structured progress events back to the window, and exposes output-opening actions for generated files.

**Tech Stack:** Python 3.11+, tkinter/ttk, Typer, Pydantic, threading + queue, ReportLab, EbookLib, PyMuPDF, pytest, ruff.

def run_engineering_books_sync(
    *,
    input_path: Path,
    output_path: Path,
    config: RunConfig,
    process_book_fn: EngineeringProcessor = process_book,
    event_listener: EventListener | None = None,
) -> list[dict[str, object]]:
    books = discover_books(input_path)
    _emit(event_listener, {"type": "run_started", "mode": "engineering", "total_books": len(books)})
    summaries: list[dict[str, object]] = []
    for book in books:
        _emit(event_listener, {"type": "book_started", "book": str(book)})
        summary = asyncio.run(process_book_fn(input_path=book, output_root=output_path, config=config))
        summaries.append(summary.model_dump() if hasattr(summary, "model_dump") else dict(summary))
        _emit(event_listener, {"type": "book_completed", "book": str(book), "summary": summaries[-1]})
    _emit(event_listener, {"type": "run_completed", "mode": "engineering", "summaries": summaries})
    return summaries
```

class GuiTaskRunner:
    def __init__(self, *, event_queue: Queue, run_engineering_fn=run_engineering_books_sync, run_publishing_fn=run_publishing_books_sync):
        self._event_queue = event_queue
        self._thread: Thread | None = None
        self._run_engineering_fn = run_engineering_fn
        self._run_publishing_fn = run_publishing_fn

    def start(self, request: RuntimeRequest) -> None:
        self._thread = Thread(target=self._run, args=(request,), daemon=True)
        self._thread.start()
```
